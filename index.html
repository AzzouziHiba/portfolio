À propos de moi

Je m'appelle Hiba Azzouzi. J'ai obtenu une licence en psychologie avec une concentration en neuropsychologie. Actuellement, je suis en Master 1 de Data Science en Santé à l'Université de Lille. Passionnée par l'analyse de données et la neuropsychologie, je cherche à allier ces deux domaines dans le cadre de projets concrets.

Mes Projets

Projet 1: Création d'une base de données sur les troubles neuropsychologiques

Dans ce projet, j'ai créé une base de données pour centraliser les informations sur les troubles neuropsychologiques chez les patients. Ce projet m'a permis de travailler sur la collecte et l'organisation des données, ainsi que sur leur analyse statistique.

Projet 2: Analyse des données cliniques pour la prévision de la dépression

En utilisant des techniques de machine learning, j'ai analysé des données cliniques pour prédire les risques de dépression chez les patients. Ce projet a impliqué la mise en place d'un modèle prédictif avec R et Python.

Projet 3: Visualisation des données de santé publique

Ce projet consistait à créer des visualisations interactives de données de santé publique pour mieux comprendre les tendances de la santé mentale dans différentes populations. J'ai utilisé des outils comme Tableau et Power BI pour ce projet.
